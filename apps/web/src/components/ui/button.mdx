# Button

A clickable element that triggers an action. All buttons use a squircle border shape via progressive enhancement (`corner-shape: squircle` on supported browsers, `border-radius: 12px` fallback).

## Usage

```tsx
import { Button } from "@/components/ui/button";

// Basic
<Button>Save</Button>

// Variant + size
<Button variant="outline" size="sm">Cancel</Button>

// As a link
<Button asChild>
  <a href="/settings">Settings</a>
</Button>
```
